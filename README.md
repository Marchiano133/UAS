# Website Company Profile - TechNova Solutions

Website company profile responsif dan interaktif yang dibuat menggunakan HTML5, CSS3, Bootstrap 5, dan JavaScript.

## Fitur Utama

1. **Responsif** - Tampilan optimal di semua perangkat (desktop, tablet, mobile)
2. **Interaktif** - Animasi dan interaksi pengguna yang menarik
3. **Greeting Otomatis** - Salam berdasarkan waktu (pagi, siang, sore, malam)
4. **Navigasi Lengkap** - 5 halaman utama dengan navigasi yang konsisten
5. **Form Kontak** - Dengan validasi client-side
6. **Filter Galeri** - Filter kategori untuk galeri foto
7. **Desain Modern** - Menggunakan Bootstrap 5 dengan kustomisasi CSS
